## Why System Design Matters for High-Paying Jobs

System Design is a critical skill for:
- **Senior Engineer roles** (80-90 LPA+)
- **Staff/Principal Engineer positions**
- **Technical Architecture roles**
- **Engineering Manager positions**

### Key Skills Tested:
1. **Scalability thinking** - Designing for millions of users
2. **Trade-off analysis** - Understanding pros/cons of each decision
3. **Problem decomposition** - Breaking complex problems into manageable parts
4. **Technology selection** - Choosing the right tools for the job
5. **Performance optimization** - Identifying and solving bottlenecks
